- Custom **Canvas + WebGL** particle systems with dynamic neural connection lines

## 🛠️ How to Use as Your Own Template

1. **Clone / fork** this repo
2. Replace the data in `lib/vercel-wizard-data.json` with your own projects (the wizard agents can generate this file for you in seconds)
3. Customize the 3D scene (colors, fog density, node styles, particle behavior) to match your brand in `src/components/Scene.tsx`
4. `npm run dev` → instant local universe
5. Deploy to Vercel — it works out of the box

This template was built for creators who ship at light speed and want their portfolio to feel like **entering their creative mind** rather than reading another list.
