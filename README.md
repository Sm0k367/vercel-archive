# Neural Archive — Sm0k367 Universe Explorer

**An immersive 3D cyberpunk neural network archive of 40+ experimental deployments.**

This is not a website.

It is a **digital twin** of the Sm0k367 creative universe — a navigable neural network where each project is a glowing node in a living cyberpunk cityscape suspended in golden-hour fog.

Built as a demonstration of what happens when human creativity + autonomous AI agents (GitHub Wizard + Vercel Wizard + Lead Architect) collaborate at maximum velocity.

## Experience It

The site features:
- **Volumetric 3D Scene** powered by React Three Fiber — floating neon monoliths, particle systems, volumetric lighting, and reactive fog.
- **Neural Connection Map** — projects are nodes. Hovering reveals connections showing iteration history (e.g. how `dj-os` evolved into `dj-smoke-stream` into `midnight-ritual`).
- **Holographic Cards** that "emerge" from the 3D space when selected.
- **The Archivist** — an AI narrator (powered by the same Grok-like persona from epic-magic) that gives lore about each project and the overall universe.
- **Live Status** from the Vercel Wizard's analysis (40 projects, 32 confirmed live, full categorization).
- **Sound-reactive elements** (music vertical projects cause the particles to pulse).

This is the evolution of the previous 2D dashboard into something that feels like stepping into the "story-book-of-portals" and "epic-magic" universe.

## Tech That Makes It Mind-Blowing

- Next.js 16 + React Three Fiber + Three.js for true 3D immersion
- Framer Motion for buttery 60fps UI transitions
- Canvas + WebGL particle systems with neural connection lines
- Real data from autonomous AI analysis (Vercel Wizard catalog + GitHub patterns)
- Consistent "golden hour cyberpunk" aesthetic across all visual systems
- Fully responsive with mobile orbit controls

## How to Use as Your Own Template

1. Clone / fork this repo
2. Replace the data in `lib/vercel-wizard-data.json` with your own projects (the wizard agents can generate this for you)
3. Customize the 3D scene colors, fog, and node styles to match your brand
4. Deploy to Vercel — it works instantly

This template is designed for creators who ship at high velocity and want their portfolio to feel like entering their creative mind rather than reading a list.
